### CR-02: Infinite loop in `generate_candidates` fallback padding for five archetypes

**Files modified:** `ideator.py`
**Commit:** `3fec047` (combined with IN-01 — both changes are tightly coupled via the `re` module)
**Applied fix:** Replaced the unbounded `while` loop that used `re.sub(r'\b5\b', ...)` (which never substitutes in skeletons without a `5`) with a new `_pad_to_min()` helper. The helper uses `_DIGIT_PATTERN = re.compile(r'\b(\d+)\b')` to find the FIRST numeric literal in the skeleton and substitutes from a fixed `_WINDOW_SUBS` list. If no digit exists, it returns early. The loop is bounded by `_WINDOW_SUBS` length and the `>= min_count` break — no infinite loop is possible.
