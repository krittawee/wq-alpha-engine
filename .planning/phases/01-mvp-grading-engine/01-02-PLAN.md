    Create sync.py. All HTTP calls use client._session (a requests.Session) against BASE_URL. Never import or call BrainClient.authenticate/login directly from sync.py — the caller owns the session lifetime.

    Imports: from wq_login import login (not used inside sync functions, only in __main__), from brain_client import BASE_URL, import db, import sqlite3, import sys.

    sync_operators(client, conn) -> int:
      Call client._session.get(f"{BASE_URL}/operators"). Call r.raise_for_status() — if 401, this raises and STOPS the run (do not catch). Parse response JSON: inspect actual shape at runtime but expect {"results": [...]} or a plain list. Normalize each item to dict with keys name, category, definition, signature (map whatever BRAIN returns; use .get() with empty-string fallback for missing fields). Call db.upsert_operators(conn, rows). Print "[sync] operators: N rows". Return len(rows).

    sync_datafields(client, conn, dataset_id="fundamental6", region="USA", universe="TOP3000", delay=1) -> int:
      Page through GET /data-fields using offset/limit pattern. Set limit=200 per page. params = {"dataset.id": dataset_id, "region": region, "delay": delay, "universe": universe, "limit": 200, "offset": 0}. Loop: GET /data-fields with params, raise_for_status, parse JSON (expect {"results": [...], "count": N} or similar), extract results list. If results is empty or len(results)==0, break. Normalize each item to dict with keys id, description, dataset, region, universe, delay, type (map from actual BRAIN field names using .get()). Accumulate rows, call db.upsert_datafields(conn, batch) per page for memory efficiency. Increment params["offset"] by len(results). After loop, print "[sync] datafields: total N rows". Return total count.

    sync_existing_alphas(client, conn) -> int:
      Page through GET /alphas with same offset/limit pattern (limit=100). For each alpha in results, build an alpha_dict mapping BRAIN's alpha JSON fields to the alphas table columns. At minimum map: alpha_id (from id or alpha_id key), expression, region, universe, delay, decay (from settings if nested), status, created_at. Set parent_alpha_id=None, archetype=None, pnl_path=None, self_corr=None, prod_corr=None, corr_checked_at=None, run_id=None, settings_json=json.dumps(alpha.get("settings",{})). Extract sharpe, fitness, turnover, returns from alpha["is"] if present, else None. Call db.upsert_alpha(conn, alpha_dict) for each. Print "[sync] existing alphas: N synced". Return total count.

    sync_all(client, conn) -> None:
      Call sync_operators(client, conn), sync_datafields(client, conn), sync_existing_alphas(client, conn) in that order. Print "[sync] complete".

    if __name__ == "__main__": guard — calls login() once, init_db(), sync_all(client, conn), then conn.close(). This lets `python sync.py` work as a standalone refresh command per the design.

    Error handling: let raise_for_status() propagate all HTTP errors including 401. Do NOT catch HTTPError and retry. For 429 with "BIOMETRICS_THROTTLED" body, print the detail and raise SystemExit (mirror wq_login.py lines 79-88 convention).

    Add import json (needed for settings_json serialization).
