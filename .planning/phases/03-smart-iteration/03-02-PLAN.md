<objective>
Build selfcorr.py — the local PnL-based self-correlation pre-filter module (ITR-03, D-08..D-13).
Implements two-stage filter: (a) proxy_gate using parent's cached PnL before simulation,
(b) precise filter post-sim before BRAIN's POST /check. Uses Python stdlib only
(no numpy). Gracefully degrades when PnL unavailable. Reads self-corr limit from
checks table at runtime — never hardcodes 0.7.

Purpose: Eliminates known-duplicate candidates without spending BRAIN API calls.
Output: selfcorr.py importable by grade.py (plan 04) and hunt.py (plan 05).
</objective>
